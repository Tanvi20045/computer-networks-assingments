📌 Assignment 1 – Network Topologies
🔹 Objectives
Design and simulate different network topologies
Compare performance and fault tolerance
🔹 Implemented Topologies
Star Topology
Hub-based Network
Ring-like Topology
🔹 Key Observations
Star topology offers better fault isolation
Hub-based networks lead to more collisions
Ring topology shows sequential data flow behavior
📌 Assignment 2 – Packet Flow Analysis
🔹 Objectives
Understand how packets move in a network
Analyze protocol behavior
🔹 Tasks Performed
Created a simple LAN using switches and PCs
Used Simulation Mode in Packet Tracer
🔹 Protocols Observed
ARP (Address Resolution Protocol)
ICMP (Ping requests & replies)
🔹 Key Learnings
MAC address table learning in switches
ARP request–reply mechanism
Step-by-step packet traversal
🧪 How to Run
Open Cisco Packet Tracer
Load any .pkt file from the assignments
Click on a PC → Desktop → Command Prompt

Run:

ping <destination IP>
Switch to Simulation Mode to observe packet flow
